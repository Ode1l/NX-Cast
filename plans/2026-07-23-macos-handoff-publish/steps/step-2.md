## Implementation Notes
Normalized the 213 Git-reported tracked text files and 49 publish candidates to
LF with a mechanical UTF-8 rewrite after proving the diff contained no binary
paths. This removed 171 EOL-only tracked modifications; the normal and
end-of-line-insensitive diff lists now match at 42 semantic paths. Removed the
verified one-byte root `NUL` file created during an earlier curl probe.

The six focused host targets passed: runtime diagnostics, network diagnostics,
protocol coordinator, AirPlay server lifecycle, AirPlay mDNS suspend, and DLNA
controller session. A clean strict Profile 14 Switch build produced a
25,600,698-byte NRO with SHA-256
`7DC90E21B382011AF824376906B124C6AFC4543B3B2B61277368FD228CB1B2B5` and both
`full-owner-exclusive-observe-bsd12` and `libnx-kernel-chacha` strings. Missing
host EGL/GLES pkg-config warnings were non-fatal for the Switch build.

The complete host `make test-airplay` was not rerun on Windows because Makefile
preflight reports `HOST_MBEDTLS_FOUND`, `HOST_SODIUM_FOUND`, and
`HOST_FFMPEG_FOUND` empty. This is an environment limitation, not a passing
claim; the handoff requires that suite after installing Homebrew dependencies on
macOS. Generated `build/`, `sdmc/`, logs, NRO, ELF, and NACP paths remain
ignored.
